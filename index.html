什么是GitHub代理？

GitHub代理是一个基于Cloudflare Workers搭建的反向代理服务，可以帮助您快速访问GitHub资源， 解决在国内访问GitHub速度慢的问题。

使用方法

1. 克隆仓库

git clone https://您的代理域名/github.com/username/repository.git

2. 下载文件

https://您的代理域名/github.com/username/repository/archive/main.zip

3. Raw文件访问

https://您的代理域名/raw.githubusercontent.com/username/repository/main/file.txt
